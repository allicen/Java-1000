<h1 class="title">Телефон (12%)</h1>
<p><b>Время: 1 сек.<br>Память: 16 Мб<br>Сложность: 12%</b></p>
<p>Услуги телефонной сети оплачиваются по следующему правилу: за разговоры до А минут в месяц – В рублей за минуту, а разговоры сверх установленной нормы оплачиваются из расчета С рублей за минуту. Требуется написать программу, вычисляющую плату за пользование телефоном для разговоров продолжительностью Т минут в месяц.</p>
<h2>Формат ввода</h2>
<p>Входной файл input.txt содержит натуральные числа A, B, C и T, не превышающие 1000. Числа располагаются в одной строке и разделены пробелами.</p>
<h2>Формат вывода</h2>
<p>В выходной файл output.txt выведите единственное целое число – месячную плату за пользование телефоном.</p>
